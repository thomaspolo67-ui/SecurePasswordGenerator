Proyecto desarrollado para la asignatura **Lógica de Programación**.

SecurePasswordGenerator es una aplicación escrita en Python que permite generar contraseñas seguras y personalizadas mediante una interfaz gráfica intuitiva. El sistema ayuda a crear contraseñas resistentes frente a ataques comunes utilizando combinaciones aleatorias de letras, números y símbolos.

---

# 📖 Descripción

Las contraseñas débiles representan uno de los principales riesgos en ciberseguridad. Este proyecto busca solucionar ese problema mediante un generador automático que permite al usuario definir las características de la contraseña y obtener un resultado seguro en cuestión de segundos.

El proyecto fue desarrollado aplicando los conocimientos adquiridos durante la asignatura de **Lógica de Programación**, utilizando estructuras condicionales, estructuras repetitivas, algoritmos y validación de datos.

---

# ✨ Características

- Generación de contraseñas aleatorias.
- Longitud configurable.
- Letras mayúsculas.
- Letras minúsculas.
- Números.
- Símbolos especiales.
- Validación de longitud mínima.
- Copiar contraseña al portapapeles.
- Interfaz gráfica desarrollada con Tkinter.

# ⚙️ Instalación

Clonar el repositorio

```bash
git clone https://github.com/thomaspolo67-ui/SecurePasswordGenerator.git
```

Ingresar al proyecto

```bash
cd SecurePasswordGenerator
```

Instalar dependencias

```bash
pip install -r requirements.txt
```

Ejecutar la aplicación

```bash
python src/main.py
```

---

# 🚀 Uso

1. Ejecutar la aplicación.
2. Seleccionar la longitud de la contraseña.
3. Elegir los tipos de caracteres.
4. Presionar **Generar**.
5. Copiar la contraseña utilizando el botón correspondiente.
